The core SSH gap: You have Crabfleet's SSH gateway for agent workspaces, but there's no persistent, secure tunnel from Claude → your Windows Server / IIS instance. The three realistic paths are: (1) a lightweight MCP server running on your Windows box exposed via ngrok or Cloudflare Tunnel, (2) GitHub OAuth so Claude can push/pull code changes without manual copy-paste, and (3) a Telegram bot as a quick notification/send channel that bridges Claude → your phone without a full integration.
