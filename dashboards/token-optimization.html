📊 Token Optimization Strategy

Pomelli AI + Design Dashboard

Document Date: March 27, 2026
Author: Jessica Muller
Purpose: Reducing AI token usage while scaling design services
Status: Knowledge Book Entry

Executive Summary

By strategically combining Pomelli AI (content generation) with Design Dashboard (client pitch tool) and Webflow/Next.js (implementation), we can reduce token consumption by 60-70% per project while delivering premium design services.

The Problem & Solution

  • Current problem: Using Claude/AI for all design work = massive token burn
  • Solution: Delegate visual/content generation to Pomelli AI, reserve tokens for strategy only
  • Result: Lower costs, faster delivery, better margins

The Token Problem (Current State)

Task Method Tokens Cost
Design descriptions (10 variations) Claude 5,000 $0.75
Layout brainstorming Claude 3,000 $0.45
Copy generation (full site) Claude 4,000 $0.60
Brand strategy refinement Claude 2,000 $0.30
TOTAL per project All Claude 14,000 tokens $2.10
⚠️ Scaling problem: 10 clients/month = 140,000 tokens = $21/month (unsustainable cost trajectory)

The Solution: Three-Tool Workflow

Tool 1: Pomelli AI (Design Generation)

  • Scans client website → extracts brand DNA → generates visual assets & copy variations
  • ✓ Token cost: $0 (separate Pomelli credits)
  • ✓ Output: 10–20 design options, hero images, social graphics
  • ✓ Time: 2–5 minutes per client

Tool 2: Design Dashboard (Client Pitch)

  • Interactive HTML tool showing 3 design directions live
  • ✓ Token cost: $0 (static HTML, no AI needed)
  • ✓ Output: Professional pitch experience, client confidence
  • ✓ Time: Client picks in real-time demo

Tool 3: Webflow or Next.js (Implementation)

  • Builds the actual website in the chosen direction
  • ✓ Token cost: Claude for custom code only (~2,000 tokens)
  • ✓ Output: Production-ready website
  • ✓ Time: 3–7 days per site

The Workflow (Step-by-Step)

Phase 1: Discovery (10 min)

  1. Client provides website URL
  2. Run Pomelli AI scan
  3. Pomelli extracts brand DNA (colors, logo, messaging, images)
  4. Download generated variations

Phase 2: Pitch (20 min)

  1. Input client info into Design Dashboard
  2. Show 3 design directions live (Professional Elegant, Premium Luxury, Bold Modern)
  3. Client picks preferred direction
  4. Document decision

Phase 3: Build (3–7 days)

  1. Use Webflow or Next.js to build final website
  2. Use small Claude prompts for custom code (~2,000 tokens)
  3. Integrate Pomelli-generated visuals & copy
  4. Deploy & launch

Phase 4: Marketing (Ongoing)

  1. Use Pomelli to generate monthly social content
  2. Reuse generated assets for email, ads, etc.
  3. Minimal Claude token use (strategy only)
